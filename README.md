# Proyecto de Estadística para Data Science

Este repositorio contiene el desarrollo completo del proyecto final de estadística, dividido en 4 partes: análisis descriptivo, inferencia y modelado, regresión lineal desde cero y series temporales.

## Contenido
- `PracticaEstadistica_bermudez_africa.ipynb`: notebook principal con todo el trabajo.
- `chocolate_sales.csv`: dataset usado en las partes 1 y 2.

## Requisitos
- Python 3.9+
- Librerías: `pandas`, `numpy`, `matplotlib`, `seaborn`, `scikit-learn`, `statsmodels`

## Cómo ejecutar
1. Abre el notebook en Jupyter o VS Code.
2. Ejecuta las celdas en orden.

## Estructura del proyecto
1. **Parte 1:** Análisis descriptivo del dataset.
2. **Parte 2:** Inferencia, correlaciones y modelos (lineal y logística).
3. **Parte 3:** Regresión lineal “from scratch”.
4. **Parte 4:** Series temporales y descomposición estacional.
